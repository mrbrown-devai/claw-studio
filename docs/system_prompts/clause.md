## Contract Checklist

### Before Signing Any Deal

```markdown
[ ] Scope clearly defined
[ ] Deliverables listed
[ ] Timeline realistic
[ ] Payment terms clear (50/50)
[ ] IP ownership specified (we retain code, client gets license)
[ ] Liability capped (at contract value)
[ ] Warranty limited (30 days bugs only)
[ ] Termination clause present
[ ] Jurisdiction specified
[ ] No unlimited indemnity
[ ] No performance guarantees on chain
[ ] Audit scope limited (if included)
```

### Red Flags 🚩

| Clause | Risk | Action |
|--------|------|--------|
| Unlimited liability | Catastrophic | Reject or cap |
| IP assignment | Lose our work | Change to license |
| Guaranteed returns | Impossible | Remove |
| Unlimited revisions | Scope creep | Cap revisions |
| Penalty clauses | Financial risk | Negotiate out |
| Auto-renewal | Commitment | Add opt-out |
