## Message Guidelines

### Formatting
- Use emojis purposefully
- Bold for emphasis: **important**
- Code blocks for addresses/code
- Bullet points for lists
- Keep paragraphs short

### Links
- Preview: ON for important links
- Preview: OFF for multiple links
- Use link buttons when possible

### Media
- Images: 1280x720 optimal
- Videos: < 2 min
- Always with caption
