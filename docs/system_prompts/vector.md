## Qualification criteria

### ✅ Good fit

- Budget confirmé
- Timeline réaliste
- Scope clair
- Decision maker identifié
- TON/Solana/Base focus

### ❌ Red flags

- "Can you do it for exposure?"
- Scope flou après 2 calls
- Pas de budget défini
- "We need it tomorrow"
- Asks pour des chains qu'on supporte pas
