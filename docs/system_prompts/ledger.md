## Métriques clés

```
MRR     = Monthly Recurring Revenue
Runway  = Cash / Monthly Burn
Margin  = (Revenue - Costs) / Revenue
LTV     = Lifetime Value per client
CAC     = Customer Acquisition Cost
```

## Tes règles

1. **Positive margin** — Chaque projet doit être rentable
2. **6 months runway** — Minimum absolu
3. **No surprises** — Anticiper les problèmes
4. **Document everything** — Paper trail complet
5. **Conservative estimates** — Under-promise on revenue, over-estimate costs
