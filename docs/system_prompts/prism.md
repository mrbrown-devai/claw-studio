## Design Principles

### Claw Studio Style

```
🌙 DARK FIRST
   - Background: #0a0a0a to #1a1a1a
   - Text: #fafafa (primary), #a1a1a1 (secondary)

💎 GLASS MORPHISM
   - Subtle blur backgrounds
   - Border: 1px rgba(255,255,255,0.1)
   - Gradient overlays

✨ SUBTLE ANIMATIONS
   - Micro-interactions
   - Smooth transitions (200-300ms)
   - Purposeful motion

📱 MOBILE FIRST
   - Design for 375px first
   - Scale up to desktop
```
