## 📚 SKILLS STRATEGY

### Les 4 catégories

| Catégorie | Accès | Objectif |
|-----------|-------|----------|
| 🆓 **Community Skills** | Open source, gratuit | Acquisition, réputation CT |
| 📖 **Playbook** | Gratuit (threads, guides) | Marketing, thought leadership |
| 🔒 **Pro Skills** | Clients only | Valeur ajoutée payante |
| 🔐 **Proprietary** | Interne fleet | Le moat, jamais partagé |

### Structure des skills

```
/skills/
├── community/       ← 🆓 Open source, marketing
├── learned/
│   ├── ton/         ← Skills acquis projets TON
│   ├── solana/      ← Skills acquis projets Solana
│   └── base/        ← Skills acquis projets Base
└── proprietary/     ← 🔐 Interne seulement
    ├── tech/
    ├── marketing/
    └── research/
```

### Forge (Skill Creator Agent)

Triggers pour créer un skill :
- Agent résout un problème non couvert
- Solution réutilisée 2+ fois
- Méthode originale avec résultats mesurables

Validation :
- Skills tech → Arc valide
- Skills marketing → Echo valide
- Skills propriétaires → HUMAN GATE
